## Welcome to Cric PR Backend API

Cric PR Backend is a production-ready cricket application API built with Node.js, Express, and MongoDB. It provides a complete suite of features for managing cricket matches, tournaments, player profiles, real-time scoring, and comprehensive statistics.

## Technology Stack

Built with modern, production-tested technologies:

- **Runtime**: Node.js (>= 12.0.0)
- **Framework**: Express.js with comprehensive middleware
- **Database**: MongoDB with Mongoose ODM (v8.18.0)
- **Authentication**: Passport.js with JWT strategy
- **Caching**: Redis & ioredis for high-performance caching
- **Real-time**: Socket.io for live match updates
- **Process Management**: PM2 with cluster support
- **Validation**: Joi for request validation
- **Documentation**: Swagger UI with OpenAPI spec

<Note>
  The API is fully documented using OpenAPI/Swagger specifications. Access the interactive API documentation at `/v1/docs` on your running instance.
</Note>

## API Architecture

The API follows RESTful principles with a well-organized structure:

```
/v1
├── /auth          # Authentication endpoints
├── /users         # User management
├── /players       # Player profiles
├── /teams         # Team management
├── /tournaments   # Tournament operations
├── /matches       # Match management
├── /innings       # Innings tracking
├── /overs         # Over management
├── /balls         # Ball-by-ball scoring
├── /stats         # Statistics and analytics
├── /notifications # Push notifications
└── /admin         # Admin operations
```

## Quick Example

Here's how simple it is to get started:

<CodeGroup>
```bash cURL
curl -X POST https://api.cricpr.com/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "password123",
    "role": "user"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.cricpr.com/v1/auth/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'John Doe',
    email: 'john@example.com',
    password: 'password123',
    role: 'user'
  })
});
const data = await response.json();
```

```python Python
import requests

response = requests.post(
  'https://api.cricpr.com/v1/auth/register',
  json={
    'name': 'John Doe',
    'email': 'john@example.com',
    'password': 'password123',
    'role': 'user'
  }
)
data = response.json()
```
</CodeGroup>
