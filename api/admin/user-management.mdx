## Dashboard Metrics

### Get Dashboard Numbers

Retrieve key platform metrics for the admin dashboard.

<CodeGroup>
```bash cURL
curl -X GET \
  'https://api.cricpr.com/v1/admin/dashboardNumbers' \
  -H 'Authorization: Bearer YOUR_ADMIN_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://api.cricpr.com/v1/admin/dashboardNumbers', {
  headers: {
    'Authorization': 'Bearer YOUR_ADMIN_TOKEN'
  }
});
```
</CodeGroup>

### Delete Team

Delete a team and all associated data.

<CodeGroup>
```bash cURL
curl -X DELETE \
  'https://api.cricpr.com/v1/admin/teams/60f7b3b3b3f3b3b3b3f3b3b3' \
  -H 'Authorization: Bearer YOUR_ADMIN_TOKEN'
```

```javascript JavaScript
const teamId = '60f7b3b3b3f3b3b3b3f3b3b3';
const response = await fetch(`https://api.cricpr.com/v1/admin/teams/${teamId}`, {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_ADMIN_TOKEN'
  }
});
```
</CodeGroup>

### Path Parameters

<ParamField path="teamId" type="string" required>
  Team ID to delete
</ParamField>

### Response

Returns HTTP 204 No Content on success.

<Warning>
  Deleting a team will remove all team instances, match associations, and may affect tournament data. This operation cannot be undone.
</Warning>
